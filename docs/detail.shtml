PTTのご案内

1. PTT とは何か

PTT とは，"Programming Tools and Techniques" の略であり，計算機・情 報関係の研究・経験談等のインフォーマルな発表会だと思っていただければ， ほぼ間違いないと思います．この会合の開催回数は 300回を越えます．PTT の 歴史は bitに掲載記事があるのでそちらをご覧下さい。

従って，発表内容に関する予稿・論文といった類のものは，全く作りませ ん．ただ，話者の方には，PTTが終った後，紙1枚くらいの簡単なメモ (これも 時々ポストしています) をお願いしております．

現在は，以下のような調子で開催いたしております．
  • 月1回(ただし，8月は休み)，主として月なかばの木曜日に開催．
  • 時間は，基本的には，18:30から20:00ごろまで．
  • 場所は，東工大(大岡山)，東大(本郷)，電通大(調布)，農工大(東小金井)，早大(新大久保)，慶応大(日吉)，の 6大学で順番に行なう．時々 「巡業」と称してこれらとは別の会場でおこなうこともある．
  • 発表の後には有志による懇親会がある．

2. PTT への参加方法

PTTには，参加資格とか参加申し込みとかいうような，面倒なものは，一切 ありません．当日，直接現地に行くだけでよいわけであります．もちろん，参 加費もなしです．

開催時間が夕食時なので，食事が必要な人は，「PTTのお知らせ」の中にあ る，``食事''のところに連絡すれば，何か適当なもの(かつどんなど)の出前を とってくれます．自分で，弁当・パンなどを買っていって，話をききながら会 場で食べるのも OK です．

3. PTT のお知らせについて

基本的にはこの Web 上で行なっていこうと思います．また，電子メールによ るサービスも行なっております．ニュースに投稿するのと同じものを希望者に直 接電子メールで送ろうというものです．希望者は，ptt-ml-admin __(at-mark)__ de.cs.titech.ac.jp（SPAM対策）まで，email をください．

4. そのほか

以上，簡単ではありますが，PTTの紹介をしました．近所の人をお誘いのう え，是非一度いらしてください．また，"このような話が聞きたい"，とか，" 某さんの話を聞きたい"という希望をどんどんお寄せいただきたいと思ってお ります．
