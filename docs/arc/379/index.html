■題目：Quine・難解プログラミングについて
■話者：遠藤侑介 (東芝)
■概要：
自分自身を出力するプログラムである Quine や、わざと
難しくプログラムを書く難解プログラミングは、古くから
プログラマの遊びとして楽しまれている。

この発表では、話者がこれまでに作った Quine や亜種、
難解なプログラムなどを、その実装技法の解説と共に紹介
する。

http://d.hatena.ne.jp/ku-ma-me/20090916/p1
http://d.hatena.ne.jp/ku-ma-me/20100905/p1
http://www.youtube.com/watch?v=r0eaf9iLKxg

多くのプログラムは Ruby で書かれており、Ruby の知識が
ある方が理解が深まるが、前提とはしない。

本発表は、話者が RubyKaigi 2010 で発表した「超絶技巧
Ruby プログラミング」を元にしており、一部の内容が重複
する。
