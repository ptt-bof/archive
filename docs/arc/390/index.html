■題目: ヘテロジニアスな計算機環境に対する自動並列化に向けて
------------------------------------------------------------------
■概要:

近年のコンピュータシステムは実効性能の向上と消費電力の低減を
同時に要求されており，それに対応するためにマルチコアプロセッ
サが一般的に利用されるようになった．さらに現在では，更なる電
力効率の向上を求められており，ＤＲＰやＧＰＵなどのアクセラレ
ータを利用するヘテロジニアスな構成をとるようになってきている．
ヘテロジニアスなコンピュータシステムでは，計算資源とプログラ
ム内の処理の関係が複雑になり，実行可否・実行コスト・データ転
送など様々な要素をプログラムを並列化する際に考慮しなくてはな
らない．本講演では，ヘテロジニアスな並列計算機環境，中でも１
チップ上に複数種類のコアを集積するヘテロジニアスマルチコアを
対象とする自動並列化手法の提案とその評価，また今後の展望につ
いて述べる．
