■題目：脳波を用いたヒトの解析〜人間中心システムに向けて〜
■概要：

近年，脳を解明し応用するための研究が盛んに行なわれている。
これらの計測には大きく3つの見地から行なわれている。
医学的見地からの解析，理学的見地からの解析，そして，工学的
見地からの解析である。これらそれぞれをさらに大分類すると，
侵襲計測・非侵襲計測に分類できる。我々は脳の中身に直接
電極を刺すことの無い非侵襲に着目し，脳波を取得することで
その解析および応用を行なっている。とくに，人間の感情などの
生理的欲求や習慣的欲求を司る前頭葉に着目し，国際10-20法に
基づくFP1箇所から得られる情報のみを用いた応用研究を紹介する。
さらに，バイオフィードバックなどの例も紹介する。
