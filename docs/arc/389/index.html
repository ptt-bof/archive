■題目: 並行制御に基づく動的記号実行拡張方式の検討
------------------------------------------------------------------
■概要:

動的記号実行（DSE: Dynamic Symbolic Execution）は，
大規模なソフトウェアを現実的な時間内で正確かつ半網羅的に
検査できる手法として，近年活発な研究対象となっている．
しかし，並列（分散）処理を検査対象とする場合，従来のDSE は
不具合の特定と再現を満足に行えない．
　この問題に対するアプローチとして，我々は従来のDSEの
枠組みを（データベースの）並行制御の要素技術で拡張する
手法を提案する．具体的には，DSE における制約処理を
トランザクションの競合解析技術で拡張することにより，
不具合の効果的特定を実現する．更に，ロールバック制御
に基づく有効範囲限定型の再実行方式により，不具合再現
の効率化を図る．
　本発表では，提案方式の概要を示し，既存手法との定性的
比較を議論する．
