■題名：計算加速器としてのGPUとそのHPCへの応用
------------------------------------------------------------------
■概要：

GPUを計算加速器として応用するGPGPUがその高い計算性能とメモリ
バンド幅のために注目されている。特にHPC分野においては、
Nvidia Teslaの東工大TSUBAMEスーパコンピュータへの導入など、
今後のさらなる利用拡大が見込まれている。本講演では初めに現在
のGPGPUの主流であるNvidia CUDAを主な対象とし、そのアーキテク
チャとプログラミングについて概略を述べる。次にGPUを実際のHPC
環境で用いる場合の課題を挙げ、それらに対する我々のグループに
よる2つの研究事例を紹介する。1点目としてGPUとCPUを併用するこ
とによってアプリケーション性能のさらなる向上を実現する手法を
紹介する。2点目としてGPUの信頼性について議論し、ソフトウェア
によって信頼性を向上させる手法を紹介する。
