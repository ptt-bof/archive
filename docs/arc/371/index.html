      ---  Programming Tools and Techniques  ---

■日時：2011年1月27日 (木) 18:30〜
■話者：渋谷 俊，高田 賢士郎（早稲田大学大学院 情報理工学専攻）
■題目：ハイブリッド制約言語HydLaのアルゴリズムの設計と実装
■概要：
ハイブリッドシステムとは時間の経過に伴って状態が連続変化したり，状態や
方程式が離散変化したりする動的システムを指す．HydLa は制約概念に基づく
ハイブリッドシステムモデリング言語であり，精度保証されたシミュレーショ
ンを行うことでシステム検証に役立てることを目標としている．本発表ではま
ずHydLa プログラムにおいて初期値が範囲を持つ場合やパラメタを用いてシス
テムを表現した場合のシミュレーションを正しく扱う非決定実行アルゴリズム
を紹介する．また現在開発しているHydLa処理系は、計算手法の1つとして数式
処理を採用しており、それによって、解軌道を時刻の式として求めたり、パラ
メタを含むモデルに対して記号的な処理を行うことが可能となっている．
HydLaの数式処理実行系における制約処理手法について述べる．また、数式処
理の特徴を活かしたシミュレーション例についても紹介する．
