■題名：
Linux カーネルにおけるエラー伝播とバグの実態調査

------------------------------------------------------------------
■概要：

Linux カーネルはサーバや組み込み機器を中心に広く使用されるようになり，
高い信頼性を要求される場面も増えてきている．一方で，linux においても信頼性
低下の要因となるバグが避けられていないことが多くの調査で示されており，
コード解析やテストなどのバグ対策が必須となる．しかし，linux はコードが複雑・
巨大で変化が速いため，現実的にはバグ対策は効率的であることが求められる．
バグ対策の効率化に向けて，バグによって発生するエラーの実態や，発生しやすい
バグの傾向を知ることは重要である．本発表では，1) linux カーネルのクラッシュ
前後の挙動をフォールトインジェクションとデバッガで追跡する調査，
2) linux カーネルにおいてよく発生していたバグの実態をパッチ説明文の自然言語処理
を用いることで分析する調査，の2つについてその方法と結果を紹介する．
