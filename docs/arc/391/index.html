■題名：
JavaRockの実装と設計，そして，次へのステップ

------------------------------------------------------------------
■概要：

FPGAは，実行したい処理を専用ハードウェア・ロジックとして実装できる
デバイスである．処理中の並列性の活用や自由なI/O接続により汎用プロ
セッサによるソフトウェア処理に比べ，高性能かつ低消費電力での実行が
期待できる．一般にFPGAのロジックはVHDLやVerilogなどのHDLによりRTLで
設計されるが，複雑な処理のRTL設計は繁雑で，バグの温床ともなる．
講演者は，複雑な処理を簡単にFPGA上に実装できるよう，Javaプログラムを
HDLに変換する高位合成処理系JavaRockを開発している．JavaRockを使うことで
設計コストの削減，また，同じ処理をソフトウェアとして実行できるためアル
ゴリズムレベルの検証コストの削減が期待できる．本講演では，JavaRockの実
装と設計，使用例を紹介する．また，これまでの開発を通じて得られた知見を
元にした次バージョンの構想について述べる．
