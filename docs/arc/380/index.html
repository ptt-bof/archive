■題目："Politics, Techniques and tools"
■概要：
現在、ITシステムの安全性を確保するための技術として、PKIを利用したシステ
ムが広く採用されている。この広範囲な利用によってシステムがかなり複雑に
なっている中、暗号アルゴリズムの危殆化からシステム全体を移行をする必要性
が出てきた。しかし、このPKI技術は、過去の経緯からネットワークが弱いこと
を前提とした設計がなされているため、現状のITシステムにマッチしているとは
言いがたい。今回は普及システムとして電子政府をとりあげ、複雑化したシステ
ムの現状と移行について解説を行う。この解説を通して、やめられないシステム
をとらざるを得ない現状を解説する。
