■題名: ICPC型プログラミングコンテストの練習による基礎コーディング技術の向上
------------------------------------------------------------------
■概要: 
ICPCの練習を通して得られるプログラムのコーディング技術につい
て述べる。ICPC(国際大学対抗プログラミングコンテスト)はACMに
より運営されている長い歴史を持つプログラミングコンテストであ
る。選手は出題される複数の問題に対し制限時間3〜5時間の間に正
しい入出力を行うプログラムを作成する。優劣の判定は解いた問題
数と解くまでにかかった時間により行われる。ICPCでは計算複雑性
、データ構造、アルゴリズムなどの情報学的な知識、応用力、記述
力が要求される。これらはコンピュータプログラミングに必要不可
欠な基礎的な力である。インターネット上には数千問に及ぶ模擬問
題が存在し、選手はこれらの問題を通して情報学の基礎を深く学ぶ
ことができる。具体的な事例として、有志の手により運営されてい
るいくつかの練習会を紹介し、その効果について述べる。
