      ---  Programming Tools and Techniques  ---
------------------------------------------------------------------
■日時：2011/07/21 (木) 18:30～
■話者：竹内郁雄 (早稲田大学理工学術院 教授, 東京大学 名誉教授)
■題目：エジプトでプログラミング言語を教える
■概要：
　そのまんまのタイトルであるが，いきなり指名を受けた竹内が，JICAの支援
で設立が進んでいる (設立した，ではない ― こういうのをsoft startという
そうだ) エジプト日本科学技術大学 (E-JUST)で，修士課程のコアコースであ
るDesign and Implementation of Programming Languages，春学期30コマを教え
た奮戦談である．
　教科書は少し迷った結果，私の手持ちの講義資料に加え，Peter van Royと
Seif HaridiのConcepts，Techniques，and Models of Computer Programming
(The MIT Press，2004) にした．終わってみて，kernel language approachを
とるこの教科書の選択はなかなかよかったと思っている．
　このあたりの話もさることながら，エジプトの政変の影響，エジプトの学生
気質，先生気質，エジプトでのカルチャ・生活に関する四方山話をからめなが
ら (むしろそちらを主にしながら)，記念すべきオール1の1バイト (8進 #377，
懐かしいpdp11のアセンブラの記法) にちなんだ楽しい時間を演出したいと考
えている．
