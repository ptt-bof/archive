話題：
 双方向変換に基づくウェブサイト更新機構 Vu-X


概要：

一般にウェブサイトは複数のウェブページから成り，情報の整合
性を保持したまま各ページを更新することは難しい．これを解決
する一つの方法として，共有の XML データとそれに対する HTML
文書への変換を複数用意することが考えられるが，この方法では
XML データを直接変更する必要があるため，限られた環境でし
か更新ができない上に，更新すべき箇所の発見が難しい．そこで，
本発表では，双方向変換言語 Bi-X を利用した新しいウェブサイ
ト更新機構 Vu-X を提案する．言語 Bi-X は，変換結果の 
編集
から変換元の更新を自動導出できるという特長を持っているため，
HTML 文書側の編集を自動的に XML データに反映させることが可
能になる．本実装では，サーバ・クライアントモデルを採用して
いるため，編集を行う端末には特別なアプリケーションをインス
トールする必要はない．また，javascriptを用いているため，
ユーザは標準的なブラウザの上でウェブページを違和感なく編集
することができ，その編集は自動的に XML データ側に反映され 
る．
