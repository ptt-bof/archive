■題目: UNICOEN: 複数プログラミング言語対応のソースコード処理フレームワーク
------------------------------------------------------------------------
■概要:

近年，プログラミング言語の多様化とソフトウェア開発を支援するソースコード
処理ツールの進歩が進んでいる．しかし，既存ツールの多くは1
つのプログラミング言語を対象として開発されているため，プログラミング言語
とツール間に多対多の関係があり，全ての言語とツールの組み合わせに対して実
装した場合，非常に莫大なコストが必要である上，ツール毎に実装や仕様に差異
が存在していて，複数のプログラミング言語で開発されたソフトウェアに適用し
づらい問題がある．
　本発表では，上述の問題を解決するために開発した，複数のプログラミング言
語に対応するソースコード処理フレームワークUNICOENについて述べる．UNICOEN
は言語非依存な言語モデルを提供することで，ツールの開発コストおよび言語の
拡張コストを削減して，ツール間の差異を低減する．UNICOEN
上で開発したツールの紹介を交えながら，UNICOENの開発を通して得られた知見
を述べる．
